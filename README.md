🐬  A Starter template built on Vite 2.x + Vue 3.x + Typescript.

**Live demo:** [Vite TS Starter](https://pdsuwwz.github.io/vite-ts-starter)

## Environment Support

* Vue 3.x
* Node >= 14.18.x

## Install

Install node dependencies in all packages

```bash
pnpm install
```

## Run

Local Development

```bash
pnpm dev
```

## Test

Unit Testing

```bash
pnpm test
```

Test code coverage

```bash
pnpm test:coverage
```
